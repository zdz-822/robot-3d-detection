# 新手：如何观看 G1 3D 检测仿真

这套仿真不会控制真实机器人。它只把一段真实 LiDAR 点云数据当作模拟 G1 的传感器输入，然后在 RViz 中显示点云和模型检测到的三维框。

## 第一次准备

第一次使用或 Docker 镜像更新后，需要打开 Ubuntu 终端，输入以下两行：

```bash
cd /mnt/c/Users/TR/Documents/Codex/2026-07-15/new-chat/robot-3d-detection
bash scripts/build_ros2_image.sh
```

看到终端最后没有 `ERROR` 即表示准备完成。这个步骤会下载图形组件，第一次可能需要几分钟；完成一次后通常不必重复。

## 每次观看

1. 在 Windows 开始菜单打开 `Ubuntu 22.04`。
2. 复制并输入下面两行，一行一行按回车：

```bash
cd /mnt/c/Users/TR/Documents/Codex/2026-07-15/new-chat/robot-3d-detection
bash scripts/run_g1_sensor_sim_rviz.sh
```

3. 等待约 20 秒，Windows 桌面会自动弹出 `RViz` 窗口。
4. 再等待约 15 秒，仿真开始播放。前两帧只用于积累时序点云，随后会看到彩色三维检测框；观看模式每帧停留约 2 秒，方便观察。

## 窗口里有什么

- 白色点：模拟 G1 LiDAR 扫描到的环境点云。
- 彩色半透明方框：模型预测到的物体三维范围。
- 方框上的文字：类别和置信度，例如 `car 0.82` 表示识别为车辆、置信度为 82%。
- 没有方框的前两帧：正常现象，模型需要三帧点云做时序融合。

## 鼠标操作

- 按住鼠标左键拖动：从不同角度看点云。
- 滚动滚轮：放大或缩小。
- 按住鼠标中键拖动：平移画面。

回放仅包含 12 帧，因此检测框会在十几秒后消失。这代表回放完成，不是程序报错。想再看一次时，关闭 RViz 窗口并重新运行“每次观看”中的两行命令。

## 如果没有弹出窗口

确认你是在 Windows 的 `Ubuntu 22.04` 终端，而不是普通 PowerShell 中运行命令。若终端显示 `ERROR`，截取最后约 20 行文字发给我即可；不要手动删除项目文件或 Docker 镜像。

如果任务栏中 RViz 图标带黄色警告且无法点开，关闭 Ubuntu 窗口后重新运行“每次观看”中的两行命令。启动脚本会自动使用 Windows 图形加速设备，不需要额外设置。
