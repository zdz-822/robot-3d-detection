# E003: BEV 检测可视化

本实验将 E002 的三帧时序模型输出为可直接展示的连续俯视图动画。

## 已完成产物

- 输入：E002 的 `checkpoint_epoch_10.pth` 权重与 nuScenes v1.0-mini 验证集同一场景的连续 12 帧点云。
- 可视化：俯视点云、青色虚线真实 3D 框、彩色实线预测 3D 框、预测类别与置信度。
- 输出：12 张 PNG、一个循环 GIF，以及记录每帧来源与预测数量的 `manifest.json`。

输出目录：`artifacts/e003_bev/`。

## 如何阅读画面

- 白色三角形：机器人当前的位置与朝向。
- 横轴：机器人前后方向的距离，单位为米。
- 纵轴：机器人左右方向的距离，单位为米。
- 青色虚线框：nuScenes 人工标注的标准答案。
- 彩色实线框：模型预测；文字例如 `car 0.82` 表示模型判断该物体是车辆，置信度为 0.82。
- 框内朝向短线：物体前方的方向。

该可视化是对 E001/E002 数值指标的直观补充，不构成新的评测指标。它证明模型可对连续点云作出稳定的 3D 检测，是 README、简历项目链接和面试展示的主要材料。
